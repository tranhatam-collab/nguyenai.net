## Chọn gói của bạn

Nguyen AI cung cấp 8 gói đăng ký (*plans*) — từ miễn phí đến doanh nghiệp. Academy miễn phí cho **tất cả** người đăng ký:

| Gói | Giá | Phù hợp với |
|---|---|---|
| *Nguyen Start* | Miễn phí | Bắt đầu khám phá — AI Office cơ bản, Nguyen Roots cơ bản |
| *Nguyen Personal* | 299K VND/tháng | Cá nhân — thêm AI Research, AI Content, Nguyen Memory |
| *Nguyen Family* | 599K VND/tháng | Gia đình — thêm AI Browser, Nguyen Knowledge, Nguyen Network |
| *Nguyen Creator* | 999K VND/tháng | Người sáng tạo nội dung — thêm AI Media |
| *Nguyen Founder* | 1.999M VND/tháng | Nhà sáng lập — thêm AI Automation, Nguyen Founders |
| *Nguyen Business* | 4.999M VND/tháng | Doanh nghiệp — thêm AI Code |
| *Nguyen Chapter* | 7.999M VND/tháng | Chapter cộng đồng — tất cả AI Tool + Nguyen Chapter OS |
| *Nguyen Enterprise* | Tùy chỉnh | Doanh nghiệp lớn — tất cả Super App + Nguyen Trust |

> Bạn có thể nâng cấp hoặc hạ cấp bất cứ lúc nào. Dữ liệu trong Data Vault và Memory được giữ nguyên khi đổi gói.

## Danh sách thiết lập

Sau khi đăng ký, làm theo **danh sách thiết lập** (*setup checklist*) sau:

### 1. Tạo tài khoản (*create account*)

- Đăng ký tại `nguyenai.net` hoặc đăng nhập qua SSO tại `app.nguyenai.net`.
- Xác minh email và số điện thoại.
- Chọn gói phù hợp (bắt đầu với *Nguyen Start* miễn phí).

### 2. Cấu hình Agent (*configure Agents*)

- Mở AI Computer Console tại `app.nguyenai.net`.
- Xem đội 9 Agent — mỗi Agent có hồ sơ (*profile*) riêng.
- Tùy chỉnh ngôn ngữ ưu tiên (tiếng Việt, tiếng Anh, hoặc song ngữ).
- Đặt mức phê duyệt (*approval level*) — mặc định nghiêm ngặt, bạn có thể nới lỏng sau.

### 3. Tải file đầu tiên vào Data Vault (*upload first files*)

- Kéo-thả một vài tài liệu gia đình vào Data Vault.
- Hoặc giao cho Nguyen Archivist: *"Lưu tài liệu này vào Data Vault"*.
- Kiểm tra quyền riêng tư — dữ liệu người đang sống phải riêng tư.

### 4. Chạy Command Pack đầu tiên (*run first Command Pack*)

- Bắt đầu với **Family Archive Pack** — đơn giản và an toàn.
- Hoặc **Founder Launch Pack** nếu bạn là nhà sáng lập.
- Theo dõi các bước, phê duyệt tại *approval gate*.
- Xem kết quả trong Data Vault với *evidence labels*.

| Ngày | Hoạt động |
|---|---|
| **Ngày 1** | Tạo tài khoản, chọn gói, mở AI Computer Console |
| **Ngày 2** | Cấu hình Agent, đặt ngôn ngữ và mức phê duyệt |
| **Ngày 3** | Tải file đầu tiên vào Data Vault, tổ chức thư mục |
| **Ngày 4** | Chạy Command Pack đầu tiên (Family Archive hoặc Founder Launch) |
| **Ngày 5** | Khám phá Super App — thử AI Research hoặc Nguyen Roots |
| **Ngày 6** | Xem lại Memory, chỉnh sửa nếu cần |
| **Ngày 7** | Hoàn thành Track 1 quiz và nhận chứng chỉ! |

## Chứng chỉ: AI Computer Fundamentals

Bạn đã hoàn thành Track 1! Bước cuối cùng:

1. **Làm bài kiểm tra** (*quiz*) — 10 câu hỏi trắc nghiệm covering lessons 1–10.
2. **Đạt điểm tối thiểu** — 7/10 câu đúng để vượt qua.
3. **Nhận chứng chỉ** (*certificate*) — với ID duy nhất, xác minh được.
4. **Chia sẻ** — dùng certificate ID để bất kỳ ai cũng có thể xác minh tại `/verify`.

> Chứng chỉ "AI Computer Fundamentals" là nền tảng — hoàn thành để sẵn sàng chuyển sang Track 2: Agent Operation.
